***Packages & Access Modifiers

A package in Java is used to group related classes. Think of it as a folder in a file directory. We use packages to avoid name conflicts, and to write a better maintainable code. Packages are divided into two categories:

Built-in Packages (packages from the Java API)
User-defined Packages (create your own packages)

-----import java.util.*;

----To create a package, use the package keyword:e.g (package packagname)


Access modifiers---define scope of variables & methods
1)public The code is accessible for all classes
2)protected--only inheritance(same package and subclasses)
3)default--- we need not to specfy the "default" word and we can access within the package.
4)private -within the class 

Access Modifiers - controls the access level
Non-Access Modifiers - do not control access level, but provides other functionality
